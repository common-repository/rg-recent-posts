Display recent posts in a widget. Specifiy which post type should be listed.

== Description ==

Display recent posts in a widget. Where the default widget only let's you display recent posts of the 'post' posttype, with RG Recent Posts you can specify from which post type you want to display the recent posts.

== Installation ==

1. Upload `RG-Recent-Posts.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You can activate the widget in the Widget section
